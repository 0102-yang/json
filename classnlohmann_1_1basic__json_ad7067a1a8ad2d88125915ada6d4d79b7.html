nlohmann::basic_json::basic_json ( const CompatibleArrayType &  value)
inline

Create an array JSON value with a given content. This constructor allows any type that can be used to construct values of type array_t. Examples include the types std::vector, std::list, and std::set.

Template Parameters
CompatibleArrayTypean object type whose value_type is compatible to array_t
Parameters
[in]valuea value for the array
Complexity
Linear in the size of the passed value.
Exceptions
std::bad_allocif allocation for array value fails (thrown by the constructor of json_value)
